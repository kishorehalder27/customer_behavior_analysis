📊 Data Analytics Project – End-to-End Pipeline
🔍 Overview

This project is a complete end-to-end data analytics workflow where I worked with a raw dataset and transformed it into meaningful business insights.

The process covers everything from data loading and exploration to building dashboards and presenting insights. The goal was not just analysis, but creating a clear story that anyone (technical or non-technical) can understand.

Python – Data loading, cleaning, and EDA

PostgreSQL – Writing and executing SQL queries
Power BI – Interactive dashboard creation
